# taxi_core

**taxi_core** — это серверная часть (ядро) системы управления такси "Taxi Luxury Service Moscow", разработанная на базе Oracle APEX.

### Состав проекта:
*   **Схема БД:** Полный набор SQL-скриптов для создания таблиц (Водители, Пассажиры, Автомобили, Заказы, Платежи и др.).
*   **Логика:** Реализация бизнес-процессов службы такси через PL/SQL.
*   **Документация:** Включена визуальная схема базы данных (схема.drawio.jpg).

### Технологии:
*   **БД:** Oracle Database.
*   **Инструментарий:** Oracle APEX, PL/SQL.
